Mention is an open-source social platform inspired by Twitter/X. It supports real-time feeds, user mentions, trending hashtags, live audio/video rooms, and ActivityPub federation — all in a single monorepo deployable to the web and mobile stores.

## Get up and running

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/OxyHQ/Mention.git
    cd Mention
    ```
  </Step>
  <Step title="Install dependencies">
    ```bash
    npm run install:all
    ```
    This installs all workspace packages and builds `shared-types` automatically via the `postinstall` script.
  </Step>
  <Step title="Configure environment">
    Copy the example environment file for the backend and fill in your MongoDB URI and any optional service keys.
    ```bash
    cp packages/backend/.env.example packages/backend/.env
    ```
  </Step>
  <Step title="Start all services">
    ```bash
    npm run dev
    ```
    This starts the Expo frontend and Node.js backend concurrently. Open the Expo dev server in your browser or scan the QR code with the Expo Go app.
  </Step>
</Steps>
